# ShopEase - E-Commerce Website

A modern, responsive e-commerce website built with React, featuring Moniepoint payment integration and a complete shopping experience.

## 🌟 Features

### Core Functionality
- **Product Catalog**: Browse and search through a variety of products
- **Shopping Cart**: Add, remove, and update product quantities
- **Secure Checkout**: Complete purchase process with Moniepoint payment integration
- **Product Search & Filtering**: Find products by category, name, or description
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices

### Pages Included
- **Home**: Hero section with featured products and company highlights
- **Products**: Complete product catalog with filtering and sorting
- **Product Details**: Individual product pages with detailed information
- **Shopping Cart**: Review and manage cart items
- **Checkout**: Secure payment process with Moniepoint integration
- **Order Success**: Order confirmation and tracking information
- **User Authentication**: Login and registration pages
- **About**: Company information and values
- **Contact**: Contact form and customer service information

### Payment Integration
- **Moniepoint Payment Gateway**: Secure payment processing
- **Multiple Payment Methods**: Bank transfer, debit/credit cards, mobile money
- **SSL Encryption**: Secure transaction processing
- **Order Tracking**: Complete order management system

## 🚀 Technology Stack

- **Frontend**: React 19 with Vite
- **Styling**: Bootstrap 5 + Bootstrap Icons
- **Routing**: React Router DOM
- **State Management**: React Context API
- **Build Tool**: Vite
- **Package Manager**: NPM

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd e-commerce-site
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173`

## 🛍️ Usage Guide

### Shopping Experience
1. **Browse Products**: Visit the Products page to see all available items
2. **Search & Filter**: Use the search bar or category filters to find specific products
3. **View Details**: Click on any product to see detailed information
4. **Add to Cart**: Click "Add to Cart" to add items to your shopping cart
5. **Manage Cart**: View and modify your cart contents on the Cart page
6. **Checkout**: Complete your purchase using the secure checkout process

### Account Management
1. **Register**: Create a new account for faster checkout and order tracking
2. **Login**: Access your account to view order history and saved information
3. **Profile**: Manage your personal information and preferences

### Payment Process
1. **Review Order**: Confirm your items and shipping information
2. **Payment Method**: Select Moniepoint as your payment provider
3. **Secure Payment**: Complete payment through Moniepoint's secure gateway
4. **Confirmation**: Receive order confirmation and tracking information
